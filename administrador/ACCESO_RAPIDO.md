## 🚀 Cómo Acceder

### Opción 1: Desde el navegador
1. Abre tu navegador
2. Copia y pega la URL: `/administrador/index.html`
3. Presiona Enter

### Opción 2: Desde la página principal
1. Inicia sesión en Bulonería Manzur
2. Busca el botón de "Administrador" o "Panel Admin"
3. Haz clic para acceder al panel completo

### Opción 3: Acceso directo desde el escritorio
1. Crea un acceso directo a: `c:\Users\ADRIAN\OneDrive\Desktop\Buloneria\administrador\index.html`
2. Haz doble clic para abrir directamente en el navegador

## ⚙️ Requisitos

- Navegador moderno (Chrome, Firefox, Safari, Edge)
- LocalStorage habilitado en el navegador
- Sesión activa en la aplicación

## 💾 Datos

Todos los datos se guardan automáticamente en el navegador usando **localStorage**.

### Datos que se guardan:
- ✅ Productos y categorías
- ✅ Movimientos de stock
- ✅ Configuración del sistema
- ✅ Sesión de usuario

## 🎯 Primeros Pasos

1. **Accede al panel** usando la URL arriba
2. **Inicia sesión** si es necesario
3. **Visualiza el Dashboard** para ver el estado general
4. **Ve a Gestión de Stock** para ver todos los productos
5. **Ajusta el stock** según sea necesario
6. **Genera reportes** para análisis

## 🔔 Notificaciones

El sistema te alertará automáticamente cuando:
- Un producto tenga bajo stock
- Se complete una operación
- Haya un error en la operación

## 🆘 Solución de Problemas

| Problema | Solución |
|----------|----------|
| No puedo entrar | Verifica que hayas iniciado sesión en la página principal |
| Los datos no se guardan | Verifica que localStorage esté habilitado |
| Página en blanco | Intenta limpiar el caché del navegador |
| Lenta | Intenta en incógnito o con menos datos cargados |
